# 层层加码 (Laymask)

一款轻量化的桌面图像蒙版合成工具——上传底图，绘制选区，将平铺图案填充到选区中，导出合成结果。

![软件截图](images\screenshot.png "软件截图")

## 功能

- **轻量化**  — 功能单一、专为“打码”而生，省去启动传统图像编辑软件
- **多种选区工具** — 矩形、椭圆、画笔、橡皮擦
- **多层编辑** — 最多 10 个独立图层，各自维护图案和参数
- **文字图案（水印）** — 自定义文字生成图案，可选字体与样式
- **模糊图案** — 对底图应用高斯模糊作为填充图案
- **历史图案** — 自动记录最近使用的图案，快速复用
- **选区编辑** — 可对选区做大小、角度、透明度等多种基础编辑
- **文件关联** — 图片文件可通过右键菜单直接打开

## 快捷键

| 按键 | 功能 |

| `E` | 椭圆选区 |

## 开发

```bash
npm install        # 安装依赖
npm run tauri:dev  # 开发模式
npm run tauri:build # 生产构建
```

## 技术栈

- **Tauri v2** — 跨平台桌面框架（Rust 后端）
- **HTML5 Canvas** — 5 层离屏画布合成管线
- **Vanilla JavaScript** — 无框架依赖
- **Vite** — 构建工具

## 项目结构

```
    ├── tauri.conf.json # 应用配置
    └── Cargo.toml      # Rust 依赖
```

## 渲染管线

```
底图 → 蒙版层(离屏A) → 图案层(离屏B) → source-in 合成 → 叠加到底图
```

1. 在主画布绘制底图
2. 在离屏画布 A 上绘制黑色选区形状（蒙版）
3. 在离屏画布 B 上用 `createPattern` 平铺填充整个画布
4. 对离屏画布 B 设置 `globalCompositeOperation = 'source-in'`，绘制离屏画布 A——图案被蒙版裁剪，PNG 透明度保留
5. 将合成结果绘制到主画布上
